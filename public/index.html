Why UNIQ ID (Demo)

Private by design

Only cryptographic commitments are stored — your paraphrase never leaves your control. The server never stores plain secrets.

Downloadable package

After registration you’ll download a small JSON package that contains your unique identifiers and signatures for offline safekeeping.

Passwordless future

Use paraphrase-derived keys and zero-knowledge friendly flows for site-specific logins in future iterations.

How it works — quick

  1. Click Register and provide your email.
  2. Verify the email via the link we send (no email is stored in readable form).
  3. Create a strong paraphrase (8–16 chars with uppercase, lowercase, digit, special).
  4. Download the generated UNIQ ID package for safe storage.
